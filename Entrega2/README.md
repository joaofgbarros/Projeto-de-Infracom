# Projeto de Infracom - Segunda Entrega
Na segunda entrega do projeto da disciplina de Infra-estrutura de Comunicação, fizemos a implementação de uma comunicação UDP confiável, aplicando o protocolo RDT3.0, apresentado no livro-texto.

### Como executar
* 1 - Execute o código para o servidor primeiro.
* 2 - Após isso, execute o código do cliente.
* 3 - Digite o nome do arquivo que você quer usar no código.
* 4 - O arquivo será modificado e o cliente receberá o arquivo modificado.
