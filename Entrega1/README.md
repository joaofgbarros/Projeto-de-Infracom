# Projeto de Infracom - Primeira Entrega
Na primeira entrega do projeto da disciplina de Infra-estrutura de Comunicação, fizemos a implementação de uma comunicação UDP utilizando a biblioteca *socket* na linguagem _python_, com envio e devolução de arquivos em pacotes de 1024 bytes.

Não foi feita a implementação de transferência confiável nesta entrega.

### Como executar
* Após baixar os arquivos, execute primeiro o ```server.py```, e após isso, o ```cliente.py```.
* Digite o nome de um arquivo no input do cliente e pressione Enter.
* O arquivo será modificado no server.py e retornado para o client.py.

### Observações
O arquivo escolhido para ser enviado deve estar na pasta ```client/``` (onde está o ```client.py```). Dentro da pasta ```client/``` já tem alguns arquivos de teste.
